Visita el sitio web

Este proyecto consistió en el desarrollo de un sitio web sencillo y funcional para una clínica podológica.
El objetivo principal era ofrecer una plataforma clara donde los pacientes pudieran conocer los servicios disponibles y contar con un acceso directo para agendar citas a través de WhatsApp.

El frontend se construyó con Astro, aprovechando su rapidez y eficiencia en la generación de páginas estáticas, mientras que los estilos se implementaron con Tailwind CSS, lo que permitió mantener una estética limpia, moderna y coherente con la identidad visual de la marca.

El resultado fue un sitio web ligero, optimizado y fácil de navegar, con secciones que destacan los tratamientos podológicos, testimonios de pacientes y un llamado a la acción constante para facilitar la comunicación con la clínica.
